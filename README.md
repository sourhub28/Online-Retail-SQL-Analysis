# 🛒 Online Retail SQL Analysis

## 📌 Project Overview

This project analyzes an Online Retail dataset using MySQL. The objective is to answer real business questions using SQL and demonstrate data analysis skills.

---

## 🎯 Objectives

- Analyze customer purchasing behavior.
- Identify top-selling products.
- Calculate revenue by country.
- Find high-value customers.
- Practice SQL for Data Analyst roles.

---

## 🛠️ Tools Used

- MySQL Workbench
- SQL
- Git & GitHub
