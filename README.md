# 🛒 Online Retail SQL Analysis

## 📌 Project Overview

This project analyzes an Online Retail dataset using **MySQL** to answer real-world business questions. The analysis focuses on customer purchasing behavior, product performance, sales trends, and country-wise revenue using SQL queries commonly used by Data Analysts.

---

## 🎯 Objectives

- Analyze customer purchasing behavior.
- Identify top-selling products.
- Calculate revenue by country.
- Find high-value customers.
- Practice real-world SQL for Data Analyst roles.

## 📊 Business Insights

The SQL analysis answered several business questions, including:

- 🌍 Which countries generated the highest revenue?
- 💰 Which invoices had the highest order value?
- 🛍️ Which products were the most expensive?
- 📦 Which products sold the highest quantity?
- 👥 Which customers placed the most orders?
- ⭐ Which customers generated the highest revenue?
- 📈 What is the average order value by country?
- 💵 What is the average product price by country?
- 🔁 Which customers are repeat buyers?
- 📦 Which products have sold more than 500 units?

---

## 📈 Key Findings

- The **United Kingdom** generated the highest number of orders and total revenue.
- Several customers placed multiple repeat orders, highlighting strong customer loyalty.
- A small number of invoices contributed significantly to total revenue.
- Product prices varied considerably across different product categories.
- High-value customers contributed a large share of overall sales.
